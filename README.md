# OneClubView

**All your clubs. One view.**

Family activity hub — track clubs, events, payments, and discover camps for your kids.

## Stack
- Frontend: Vanilla React (single HTML file)
- Backend: Supabase (Postgres, Auth, Edge Functions)
- Hosting: Netlify
- AI: Claude API for camp scraping

## Deploy
- Publish directory: `public`
- No build command required
